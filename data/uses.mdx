### Tech

- [React](https://reactjs.org/) - La biblioteca frontend más utilizada para crear interfaces de usuario.
- [nextjs](https://nextjs.org/) - Buscas experiencia al desarrollar aplicaciones web y mas.
- [Prisma](https://www.prisma.io/) - ORM de TypeScript y Node.js de próxima generación (Lo uso con Postgres alojado en supabase.io).
- [Testing Library](https://testing-library.com/) - Una gran biblioteca de pruebas para cualquier cosa que interactúe con el DOM.
- [Jest](https://jestjs.io/) - Jest es un marco de pruebas de JavaScript delicioso que se centra en la simplicidad.
- [MSW ](https://mswjs.io/) - API burlándose de la próxima generación
  (Simulacros interceptando solicitudes a nivel de red)
- [Prettier](https://prettier.io/): Formateador de código. Me encanta.
- [supabase](https://supabase.io/) - Firebase Alternative.
- [firebase](https://firebase.google.com/) - Firebase te ayuda a compilar
  y ejecutar apps exitosas.
- [mdx-bundler](https://github.com/kentcdodds/mdx-bundler): Compila y agrupa tus archivos MDX. Me encanta.
- [radix-ui/stitches](https://stitches.dev/): Me ha dado una experiencia increíble diseñando mis componentes.
- [radix-ui/colors](https://www.radix-ui.com/colors): Execelentes y accesibles sistema de colores.
- [radix-ui/icons](https://icons.modulz.app/): Execelentes y nítidos iconos.
